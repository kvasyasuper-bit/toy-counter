# Учёт игрушек в детском саду

Веб-приложение для детекции и подсчёта игрушек на изображении с помощью
предобученной нейросети **YOLOv8** (датасет COCO).

Проект выполнен в рамках производственной практики (Системы искусственного
интеллекта), направление 09.03.01 «Информатика и вычислительная техника».

## Возможности

- Загрузка изображения через веб-интерфейс.
- Кнопка «Запустить обработку».
- Детекция игрушек и отрисовка bounding boxes.
- Вывод обработанного изображения и статистики (количество объектов).
- Возврат статистики в формате JSON.
- Сохранение истории запросов в базе данных SQLite.
- Выгрузка отчёта в форматах Excel и PDF.

## Стек технологий

- Python 3.10+
- Flask — веб-сервер и REST API
- Ultralytics YOLOv8 — нейросеть детекции объектов
- OpenCV — чтение изображений и отрисовка результатов
- SQLite — хранение истории
- openpyxl / reportlab — генерация отчётов

В стандартном датасете COCO отсутствует общий класс «игрушка», поэтому
приложение учитывает ближайшие подходящие классы (`teddy bear`, `sports ball`,
`frisbee`, `kite`). Для промышленного применения модель следует дообучить на
собственном наборе размеченных изображений игрушек.
